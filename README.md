# Interactive structure viewer
This repository provides an interactive web visualization of two of the Pd<sub>4</sub>Au<sub>16</sub>/CeO<sub>2</sub>(111) model structures reported in E. Fernández-Villanueva, P. J. Ramírez, P. G. Lustemberg, R. Pérez, M. V. Ganduglia-Pirovano, J. A. Rodriguez, Angew. Chem. Int. Ed.2025, 64, e202505716.
DOI: https://doi.org/10.1002/anie.202505716.

The viewer is implemented with 3Dmol.js and was created for convenient inspection of the structures on desktop and mobile devices.
